# Mapa de Imagens do Projeto CNH de Novo

Todas as imagens usadas no projeto estão na pasta `/public/assets/`.

| Arquivo                  | Usado em              | Descrição                        |
|--------------------------|-----------------------|----------------------------------|
| `logo-navbar.png`        | Navbar.tsx            | Logo exibida na barra de navegação |
| `logo-footer.png`        | Footer.tsx            | Logo exibida no rodapé           |
| `hero-image.jpg`         | HeroSection.tsx       | Imagem de fundo da seção hero    |
| `logo.png`               | (backup)              | Logo original do projeto         |

## Ícones
Todos os ícones são renderizados via **Lucide React** (tree-shakable SVGs).
Não há dependência de ícones externos ou CDN.
